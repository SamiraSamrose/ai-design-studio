# User Guide

## Getting Started

### First Design

1. Launch application
2. Enter product description
3. Select product type
4. Click "Generate Design"
5. Wait for generation (30-60 seconds)
6. View result in preview panel

### Understanding Parameters

- **Camera Angle**: Perspective of product view
- **FOV**: Field of view (wider = more context)
- **Lighting**: Illumination style
- **Material**: Surface properties
- **Reflectivity**: How shiny the surface appears
- **Texture Quality**: Detail level
- **HDR**: High dynamic range processing
- **Bit Depth**: Color precision (16-bit for production)
